## Источники данных

### Jhani & Pandit, 2022 (описательный опрос)
Набор данных, собранный путём опроса в Google Forms Хитешу Джани и Канаком Пандитом, студентами инженерно-технологического колледжа Такура в Мумбаи, Индия. Файл содержит **244 записи**, после фильтрации по стране — **235 респондентов**. Анкета из 22 вопросов охватывает регулярные оздоровительные и спортивные практики, барьеры и мотивацию, а также отношение к онлайн-сервисам тренировок.

Ссылка: [Mendeley Data](https://data.mendeley.com/datasets/s3n589mpzc/1).

### Research on Online Sports Services (Ответы), 2026

Опрос в Google Forms, проведённый в феврале–марте 2026 года среди жителей Индии. Анкета основана на расширенной модели принятия технологий **UTAUT2** и включает 7 конструктов-предикторов и 1 целевую переменную, каждый измеряется тремя пунктами по 5-балльной шкале Ликерта:

| Код  | Конструкт                    | Роль               |
|------|------------------------------|---------------------|
| PU   | Воспринимаемая полезность    | предиктор           |
| PEOU | Лёгкость использования       | предиктор           |
| SI   | Социальное влияние           | предиктор           |
| FC   | Поддерживающие условия       | предиктор           |
| HM   | Гедоническая мотивация       | предиктор           |
| PV   | Ценность цены                | предиктор           |
| HAB  | Привычка                     | предиктор           |
| BI   | Поведенческое намерение      | целевая переменная  |

Полная выборка — **186** респондентов. Аналитическая подвыборка (исключаем респондентов, не занимающихся и не собирающихся заниматься спортом) — **164** респондента.

## Описательный анализ

### Jhani & Pandit (`jhani_pandit_analysis/notebooks/`)

`01_data_cleaning` — фильтрация респондентов не из Индии, нормализация названий городов, замена одинаковых по смыслу ответов на единую категорию, представление множественнового выбора в виде бинарных индикаторов. Результат — `Fitness_Survey__Responses_clean.xlsx` (235 × 52).

Ноутбуки `02`–`10` содержат визуализации распределений ответов: демография и география выборки, типы и частота физической активности, три блока барьеров (к тренировкам, питанию, онлайн-сервисам), цели занятий, готовность к онлайн-формату и критерии выбора поставщика.

### Собственный опрос (`utaut2_analysis/notebooks/`)

`02_utaut2_subsample` — расчёт 8 конструктов UTAUT2, формирование аналитической подвыборки (164 из 186). Результат — `utaut2_subsample.xlsx` (164 × 45).

Ноутбуки `03`–`05` содержат визуализации распределений по демографии, текущей физической активности, опыту онлайн-фитнеса, целям и барьерам.

## Модель UTAUT2 (`utaut2_analysis/notebooks/`)

#### `06_utaut2_descriptive.ipynb`

Описательные статистики и boxplot распределений восьми конструктов.

**Результат:** наибольшее среднее — у FC (M = 4.16), наименьшее — у HAB (M = 3.01). HAB показывает наибольший разброс (SD = 1.21). BI = 3.71 — респонденты в целом скорее готовы использовать онлайн-фитнес.

#### `07_utaut2_reliability.ipynb`

Значения коэффициента Альфа-Кронбаха для каждой шкалы.

**Результат:**

| Конструкт | Cronbach's α |
|-----------|-------------|
| HAB       | 0.895       |
| PV        | 0.698       |
| BI        | 0.664       |
| SI        | 0.651       |
| PU        | 0.634       |
| HM        | 0.515       |
| PEOU      | 0.414       |
| FC        | 0.132       |

Высокую надёжность (α > 0.7) демонстрирует только HAB. Низкие значения FC (0.132) и PEOU (0.414) указывают на слабую внутреннюю согласованность этих шкал.

#### `08_utaut2_correlations.ipynb`

Коэффициенты ранговые корреляции Спирмена предикторов с целевой переменной BI.

**Результат:**

|          Конструкт            | ρ      | p-value   |
|-------------------------------|--------|-----------|
| Воспринимаемая полезность (PU)| 0.584  | < 0.0001  |
| Привычка (HAB)                | 0.434  | < 0.0001  |
| Гедоническая мотивация (HM)   | 0.306  | < 0.0001  |
| Социальное влияние (SI)       | 0.297  | < 0.0001  |
| Лёгкость использования (PEOU) | 0.296  | < 0.0001  |
| Ценность цены (PV)            | 0.194  | 0.0130    |
| Поддерживающие условия (FC)   | 0.092  | 0.2430    |

Наиболее сильная связь с поведенческим намерением у воспринимаемой полезности и привычки. Поддерживающие условия единственный конструкт без значимой корреляции при alpha = 0.05.

#### `09_utaut2_group_comparisons.ipynb`

Критерий Краскела-Уоллиса по группам опыта использования онлайн-фитнеса, Манна-Уитни по полу и возрасту.

**Результат:**
Критерий Краскела-Уоллиса подтверждает статистически значимые различия между этими группами (H = 14,31, p < 0,001). Те, кто пользуется онлайн-сервисами сейчас (M = 3.92), показывают значимо более высокое BI, чем никогда не пользовавшиеся (M = 3.47). Различий по полу и возрастным группам выявлено не было.

#### `10_utaut2_normality.ipynb`

Shapiro–Wilk, асимметрия и эксцесс для каждого конструкта.

**Результат:** нормальность отвергается для всех конструктов (p < 0.01), но асимметрия (|skew| < 0.6) и эксцесс (|kurt| < 1.7) остаются в умеренных пределах, что допускает OLS при n = 164.

#### `11_utaut2_regression.ipynb`

OLS-регрессия `BI ~ PU + PEOU + SI + FC + HM + PV + HAB`, стандартизированные β*.

**Результат:** R² = 0.530, Adj. R² = 0.508, F = 25.09 (p < 0.001) — модель объясняет 53% дисперсии BI.

| Предиктор | β*    | p-value | Значимость |
|-----------|-------|---------|------------|
| PU        | 0.415 | < 0.001 | ***        |
| HAB       | 0.307 | < 0.001 | ***        |
| HM        | 0.191 | 0.004   | **         |
| PEOU      | 0.129 | 0.042   | *          |
| FC        | 0.067 | 0.245   | н.з.       |
| SI        | 0.067 | 0.294   | н.з.       |
| PV        | 0.005 | 0.931   | н.з.       |

Ключевые предикторы — PU (β* = 0.415) и HAB (β* = 0.307). SI, FC и PV не вносят значимого вклада при контроле остальных.

#### `12_utaut2_residuals.ipynb`

Диагностика остатков: остатки vs предсказанные, Q-Q plot, Shapiro–Wilk.

**Результат:** Диагностика остатков не указывает на критические нарушения предпосылок линейной регрессии. Распределение остатков не отличается от нормального на стандартном уровне значимости.
